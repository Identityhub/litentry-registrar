# litentry-registrar
Litentry Polkadot/Kusama Registrar

Install the dependencies

```
npm install .
```

Setup the environment

```
echo 'NODE_ENV=dev' > ./.env
mkdir -p ./log/litentry-registrar
```

Start the server

```
npm run app
```
